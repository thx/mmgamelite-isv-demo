## 使用方法

- 下载并安装 [淘宝开发者工具](https://developer.taobao.com/)

- 下载demo代码

```bash
# 克隆代码
~ git clone https://github.com/thx/mmgamelite-isv-demo.git

# 安装依赖
~ cd mmgamelite-isv-demo 
~ npm install
```

- 使用 `淘宝开发者工具`打开demo项目。
    - 项目类型选择`模板`
    - 选择相应`关联应用`
    - 进行开发调试和预览。
